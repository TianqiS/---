### js相关

- js拓展运算符
  
  扩展运算符内部调用的是数据结构的 Iterator 接口，因此只要具有 Iterator 接口的对象，都可以使用扩展运算符，比如 Map 结构。
  
  如果对没有iterator接口的对象，使用扩展运算符，将会报错。
  
  ```javascript
  var obj = {a: 1, b: 2};let arr = [...obj]; // TypeError: Cannot spread non-iterable object
  ```
